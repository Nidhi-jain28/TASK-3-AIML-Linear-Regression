# AI/ML Internship Task 3 - Linear Regression

## Objective
The objective of this task was to understand and implement linear regression using a house price prediction dataset.

## Dataset Used
- House Price Prediction Dataset

## Tasks Performed
- Loaded and explored the dataset
- Preprocessed categorical variables
- Split the dataset into training and testing sets
- Implemented Multiple Linear Regression using Scikit-learn
- Predicted house prices
- Evaluated the model using MAE, MSE, and R² score
- Visualized actual vs predicted house prices

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## Learning Outcome
Through this task, I learned the basics of regression modeling, data preprocessing, model training, prediction, and evaluation. I also understood how multiple features contribute to predicting a target variable and how model performance can be measured using different evaluation metrics.
